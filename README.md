# Secret Message Encoder

A Java application that encodes and decodes secret messages using string manipulation techniques.

## Overview

This project implements a secret message encoding system that transforms plain text into encoded strings and back. It demonstrates string processing, character manipulation, and utility class design in Java.

## Features

- Encode plain text messages into secret format
- Decode secret messages back to readable text
- Reusable string utility helpers
- Test suite covering encode/decode operations

## Technologies Used

- **Language:** Java (JDK 8+)
- **Build Tool:** Apache Ant (NetBeans)

## Project Structure

```
src/
├── StringUtility.java    # Core string manipulation helpers
├── SecretMessage.java    # Encoding and decoding logic
└── MessageTest.java      # Test cases and main entry point
```

## Getting Started

```bash
javac src/*.java -d out/
java -cp out/ MessageTest
```
